
# Maus Ray Tracing in One Weekend

I made this project in rust following the [Ray Tracing in One Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html) Book. 

## Running
```sh
cargo run -qr > image.ppm
```
